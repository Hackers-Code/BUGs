0x0 - exit (both)
0x1 - connect (client)
0x2 + 4 bytes (id) - accept connection (server)
0x3 + 4 bytes (id) + 20 bytes (nickname) - reserve nickname (client)
0x4 + bool ( nickname status ) - accept nickname (server)
0x5 - list games (client)
0x6 + 4 bytes (games count) + (4bytes (game id) + 20 bytes (game name)) * games count - list games ( server )
0x7 + 4 bytes (id) + 20 bytes (game name) + 1 byte ( password_length ) + password_length bytes (password ) - create game (client)
0x8 + bool ( game name status ) - accept game creating (server)
