0x0 - kick from server ( server, TCP )
0x1 - removed
0x2 - removed
0x3 + 20 bytes (nickname) - reserve nickname ( client, TCP )
0x4 + 1 byte (status) - nickname status ( server, TCP )
0x5 - list games ( client, TCP )
0x6 + 4 bytes (games count) + {4bytes (game id) + 20 bytes (game name)} * [games count] - list games ( server, TCP )
0x7 + 20 bytes (game name) + 1 byte (password length) + [password length] bytes (password) - create game ( client, TCP )
0x8 + 1 byte (status) - creating game status ( server, TCP )
0x9 + 4 bytes (map id) + 1 byte (players_count must be equal 2,3 or 4) - set game config ( client, TCP )
0xa + 1 byte (status) - game config status ( server, TCP )
0xb + 1 byte (color) + 1 byte (mask) - set character properties ( client, TCP )
0xc + 1 byte (status) - are properties unique ( server, TCP )
0xd - reserved
0xe - reserved
0xf - reserved
0x10 + 4 bytes (game id) + 1 byte (password length) + [password length] bytes (password) - join game ( client, TCP )
0x11 + 1 byte (status) + 4 bytes (mapID if status equal 1) - accept join game ( server, TCP )
0x12 - switch confirm joining game ( client, TCP )
0x13 - confirm that server received confirmation ( server, TCP )
0x14 - all players confirmed, load the map ( server, TCP )
0x15 - get players in game ( client, TCP )
0x16 + 4 bytes (players count) + {1 byte(id) + 20 bytes(name)}*[players_count] - list players in game ( server, TCP )
0x17 - get worms positions and health ( client, UDP )
0x18 + 4 bytes (worms_count) + {1 byte(owner_id)+4bytes(x)+4bytes(y)+1byte(hp)+1byte(worm_id)}*worms_count - list living worms ( server, UDP )
0x19 + 1byte (worm_id) - you have 60 seconds to move ( server, UDP )
0x1a - how much time left ( client, UDP )
0x1b + 1 byte (seconds) - time left ( server, UDP )
0x1c - time is out, did damage or used weapon, end move ( server, UDP )
0x1d - jump ( client, UDP )
0x1e - switch move left ( client, UDP )
0x1f - switch move right ( client, UDP )
//error opcodes
0xe0 - unknown opcode ( server, TCP )
0xe1 - bad parameters ( server, TCP )
0xe2 - server error ( server, TCP )
0xe3 - unexpected opcode ( server, TCP )
